Internet Sources

Rapid Prototyping of Application Specific Signal Processors (RASSP), DARPA [Online]. Available: http://www.people.vcu.edu/~rhklenke/tutorials/vhdl/modules/m10_23/sld001.htm
An excellent free resource for VHDL related educational material. Several descriptions and figures in Section 3 have been based on slides in Module 10 - VHDL Basics.
Peter J. Ashenden, The VHDL Cookbook, first ed, 1990 [Online]. Available: http://esd.cs.ucr.edu/vhdlcook/
This is a basic introduction to the VHDL language that is useful for looking up language constructs and syntax. It does not cover synthesis concepts, and should really only be used as a language reference. It is available for free.
Compact Summary of VHDL v1.5, Language reference from the University of Maryland, Baltimore County (UMBC) [Online]. Available: http://www.csee.umbc.edu/portal/help/VHDL/summary.html
Succinct language reference manual useful for looking up syntax.
Jim Lewis (2003), "VHDL Math Tricks of the Trade," Tutorial at Military and Aerospace Programmable Logic Devices (MAPLD) Conference [Online]. Available: http://www.gstitt.ece.ufl.edu/vhdl/refs/vhdl_math_tricks_mapld_2003.pdf
Excellent summary on available VHDL libraries, packages and types to use for numeric operations, and related techniques. Comprises just a few slides but very useful.
Vivado Design Suite User Guide [Online]. Available: https://docs.xilinx.com/r/en-US/ug910-vivado-getting-started/Vivado-Design-Suite-Overview
Getting started with Vivado integrated design environment (IDE).
Vivado Simulator Overview [Online]. Available: https://docs.xilinx.com/r/2021.2-English/ug937-vivado-design-suite-simulation-tutorial/Vivado-Simulator-Overview
Getting started with Vivado simulator.
Archive of VDLANDE.com VHDL Reference, Language reference from Jayram Moorkanikara at University of California Irvine [Online]. Available: https://blog.eowyn.net/vhdlref/websites/jmoorkan/vhdlref/
Succinct language reference manual useful for looking up syntax.

Selected Text Books

Douglas L Perry, VHDL Programming by Example, 4th ed. McGraw-Hill, 2002.
Excellent textbook with comprehensive treatment of both language aspects and synthesis.
Douglas J. Smith, HDL Chip Design: A practical guide for designing, synthesizing and simulating ASICs and FPGAs using VHDL or Verilog, Doone Publications 1998.
A good book that succintly describes VHDL and also covers synthesis aspects. Includes VHDL and Verilog code side-by-side.
J. Bhasker, A VHDL Synthesis Primer, 2nd ed. Star Galaxy Pub., 1998.
Provides a cook-book type introduction to synthesis.
